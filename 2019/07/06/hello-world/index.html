你好，Hexo ！

发表于 2019-07-06 更新于 2022-07-08 分类于 搭建博客 评论数： 0 阅读次数：

  学一门编程语言，新使用一个软件第一个demo都是 “你好，世界！” ，循此例，第一篇文章就叫“你好，Hexo ！”。后续会把其它博客搬到此处。

博客折腾记

  刚开始接触编程时就喜欢瞎捣鼓，那时到处蹭免费的vps，像AWS，Linode，IBM Bluemix都折腾过。最开始使用WordPress搭建，后来到期没续费，很多数据都弄丢了。再后来自己用Python写的程序放在阿里云上，简单，速度快，但是由于经常在阿里云测试些新玩意儿，服务不是很稳定。最近了解到GitHub提供免费的GitHub Page功能可以搭建博客，并且有个非常流行的博客框架Hexo。折腾两天，基于GitHub + Hexo的博客上线了。后续会稳定在这个上写作。

博客感想

  最近总在想人有什么是值得去做的，每天忙碌，忘我的工作，独立思考的时间很少，慢慢的变得随波逐流，追逐各种东西。但对于短短几十年的人生该过成什么样？有什么为之努力奋斗使之有意义，能在自己百年之后骄傲的说声“我来过！”的事情似乎没有。大多数人都是极其平凡的度过人生这几十年的光景，出生，上学，工作，生儿育女，步入老年，死去，下一代又重复着我们的轨迹。能在历史长河中留下浪花的人都是人中龙凤，绝大多数的人都是平凡的一滴小水珠。
  在网络上留下个小小的空间记录下生活中的点点感悟，也证明自己曾来过这个世界。。。
