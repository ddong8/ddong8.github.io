优势

Jetbrain家的pycharm也支持远程解释器，但是其原理为本地，远程主机上代码映射，隔段时间同步代码，并不是真正的在远程主机上开发。VS Code则是直接编辑服务器上的文件，同时能自动匹配开发语言环境，直接可以执行，调试相当方便！
